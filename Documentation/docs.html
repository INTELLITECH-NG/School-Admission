How to Setup - Automatic



Step1: Upload Files.zip to your host and unzip.
Step2: Create mySQL Database and Database User.
Step3: Open your browser and visit http://your-sitename/install_glix.
Step4: Fill All Information there and click SETUP.
Step5: System is ready, you able to use now.
More: If still you failed to install use the manual installation process.


How to Setup Manually



Step1: Upload Files.zip to your host and unzip.
Step2: Create mySQL Database and Database User.
Step3: Go to your root folder and locate db_params.php
Step4: Replace DB Name DB User and DB Password with your Hosting settings.
Step5: Import glix_db.sql to your Phymyadmin
Step6: Setup Complete, login and remember to change your password. You can Hash your password incase you want to.
Login details: Username: admin Password: admin


How To Edit Code?


Once You Install The Product on Your Server, You Will have Eight Folder and Few Files.

On Folder "admin", We keep all the admin functions inside this folder, such as the Student Pictures, Generated Pins, Exams Links. So if You Want To Edit Or Change Any Code, Have a Look on This Folder.

On Folder "CSS", We keep FONTS, PLUGINS, BOOTSTRAP.

We Maintain Pure PHP Coding on This Project.

"inc/config.php" All the system settings such as the App name, App Version, App Date are kept in this folder.

"inc/page_footer.php" All page footers are in This Folder.

"inc/page_head" All project headers such as Dashboard controllers and Headers links are in the folder.

"inc/template_start.php" You Have All HTML Structure Here. You can Change anything you want if you need to change anything on HTML.

"inc/template_end.php" You Have All HTML Structure Here. You can Change anything you want if you need to change anything on HTML.

"inc/template_scripts.php" You Have All Script Src Here. You can Change anything you want if you need to change anything in the Js.

PHP Structure


The PHP Structure uses version 5, which the system config can be found in the inc/config.php. This is where you change the frontend header links on the menu tab.

CSS Files and Js Structure


Mainly three main CSS files are used in this theme. The first one bootstrap.css, second one is style.css which for this template and third one is responsive.css to control responsive layouts.

Used Css File For Various Purpose :

bootstrap.min.css (in css Folder)

main.css (in css Folder)

plugins.css (in css Folder)

themes.css (in css Folder)

app.js (in js Folder)

helpers, pages, vendor (in js Folder)

fonts, js, main, plugins, themes, bootstrap.less, styles.less, themes.less, variables.ledd (in less Folder)


If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.
